# Linux Learning Journal

This repository documents my journey learning Linux while building and maintaining my Arch Linux workstation and home lab.

I record problems I encounter, how I debug them, and what I learn from fixing them.

Topics include:

- Arch Linux configuration
- shell configuration and troubleshooting
- networking and SSH
- printer setup
- useful scripts and automation
- general Linux experiments

The goal is to build a record of my learning and create a reference I can revisit later.
